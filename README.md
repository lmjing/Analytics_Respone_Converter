# Analytics_Response_Converter
Analytics API Response를 하이차트 데모 코드에 쉽게 적용가능하도록 바꿔주는 간단한 웹

# 사용법
1. 웹 페이지 띄우기
`git clone` 받은 파일 내 test.html을 더블 클릭해 웹 페이지를 띄워줍니다.

2. API 결과 폼에 넣기
http://localhost:8180/analyticsapi/swagger#!/stat-controller/selectCommonStatUsingPOST 에서 직접 테스트하여 나온
Response Body를 모두 복사하여 넣어줍니다. (성공일 경우에만)
